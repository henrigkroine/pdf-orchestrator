## Accessibility Matrix

### Excellent Contrast (WCAG AAA - 7.0+)
```
✅ Nordshore on White      12.8:1    Perfect for all text
✅ Nordshore on Sand        8.2:1    Perfect for all text
✅ White on Nordshore      12.8:1    Perfect for all text
✅ Black on White          21.0:1    Perfect for all text
✅ Black on Sky            15.8:1    Perfect for all text
```

### Good Contrast (WCAG AA - 4.5+)
```
✅ Moss on White            4.8:1    Good for normal text
✅ Clay on White            6.1:1    Good for normal text
✅ Clay on Sand             4.2:1    Good for normal text
```

### Large Text Only (WCAG AA Large - 3.0+)
```
⚠️ Gold on White           3.2:1    18pt+ or 14pt+ bold only
⚠️ Moss on Sand            3.7:1    18pt+ or 14pt+ bold only
```

### Avoid
```
❌ Gold on Sand            2.1:1    Too low contrast
❌ Sky on White            1.8:1    Too low contrast
```

## Image Overlays

### Dark Overlay (White Text on Photos)
```
Color:       Nordshore (#00393F)
Opacity:     40-60%
Blend Mode:  Multiply
CSS:         background: rgba(0, 57, 63, 0.4);
Use:         Hero images, photo backgrounds
```

### Light Overlay (Dark Text on Bright Photos)
```
Color:       Sand (#FFF1E2)
Opacity:     30-50%
Blend Mode:  Screen
CSS:         background: rgba(255, 241, 226, 0.3);
Use:         Toning down bright photos
```

### Gradient Overlay (Hero Images)
```
Colors:      Nordshore → Transparent
Direction:   To bottom
Opacity:     60%
CSS:         linear-gradient(to bottom, rgba(0,57,63,0.6), transparent);
Use:         Hero sections with text
```

## Quick Reference Commands

### JavaScript
```bash
# Get color info
node color-intelligence.js palette partnership_document

# Validate color
node color-intelligence.js validate "#00393F"

# Check contrast
node color-intelligence.js contrast nordshore white

# Export CSS
node color-intelligence.js css partnership_document > colors.css
```

### Python
```bash
# Get color scheme
python color_harmony.py scheme partnership_document

# Validate color
python color_harmony.py validate "#C87137"

# Check contrast
python color_harmony.py contrast nordshore sand

# Apply to InDesign
python apply_colors_intelligent.py partnership_document
```
