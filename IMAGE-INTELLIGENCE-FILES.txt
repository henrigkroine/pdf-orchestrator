CORE FEATURES
=============

1. Intelligent Image Sizing - Automatic sizing to fill frames without distortion
2. Logo Clearspace Enforcement - TEEI brand compliance (clearspace = logo height)
3. AI Hero Generation - DALL-E 3 with TEEI brand prompts
4. Brand Overlays - Apply Nordshore/Sky/Sand tints at correct opacity
5. Intelligent Cropping - Golden ratio, rule of thirds, AI-guided
6. Quality Optimization - 300 DPI print, 150+ digital, web
7. Batch Processing - Efficient multi-image operations

CLI COMMANDS
============

node image-intelligence.js generate "concept"          # Generate hero
node image-intelligence.js optimize image.jpg print    # Optimize
node image-intelligence.js overlay image.jpg nordshore 0.4  # Overlay
node image-intelligence.js crop image.jpg golden-ratio # Crop
node image-intelligence.js clearspace logo.png         # Clearspace
node image-intelligence.js stats                       # Statistics

JAVASCRIPT API
==============

const ImageIntelligence = require('./image-intelligence.js');
const imageAI = new ImageIntelligence();

await imageAI.placeImageInFrame(imagePath, width, height, options);
await imageAI.generateHeroImage(concept, options);
await imageAI.applyBrandOverlay(imagePath, color, opacity);
await imageAI.intelligentCrop(imagePath, { strategy: 'golden-ratio' });
await imageAI.calculateLogoClearspace(logoPath, { createGuide: true });
await imageAI.optimizeImageQuality(imagePath, 'print');
await imageAI.batchProcess(imagePaths, operation, options);
