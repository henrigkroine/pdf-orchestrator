## 1. COVER PATTERN (TFU_Cover)

**Used on**: All Together for Ukraine program overview covers

### Layout Structure
```
┌─────────────────────────────────────────────┐
│ [TEEI Logo - white]                         │ ← Top-left, ~40pt from edges
│                                             │
│                                             │
│     ┌───────────────────────────────┐      │
│     │                               │      │
│     │   [Hero Photo Card]          │      │ ← Large rounded rectangle
│     │   Warm, authentic photo       │      │   Centered, ~60-70% page width
│     │   Rounded corners: ~24pt      │      │   Height: ~50-60% page height
│     │                               │      │
│     └───────────────────────────────┘      │
│                                             │
│                                             │
│        Together for Ukraine                │ ← Large serif, white, centered
│   PROGRAM SUBTITLE IN SMALL CAPS           │ ← Sans serif caps, white, centered
│                                             │
└─────────────────────────────────────────────┘
```

### Specifications
- **Background**: Deep teal #00393F (RGB 0, 57, 63) - full bleed
- **TEEI Logo**: White 3-bar logo, top-left corner (~40pt margins)
- **Hero Photo Card**:
  - Dimensions: ~460pt wide × 400-450pt tall (on 612pt page width)
  - Position: Centered horizontally, ~210pt from top
  - Rounded corners: 24pt radius
  - Photo: Warm, authentic, human-focused (refugees, families, students)
  - Photo treatment: No overlay, clean rounded card
- **Document Title**:
  - Font: Lora Bold or similar serif
  - Size: 56-64pt
  - Color: White #FFFFFF
  - Position: Centered, ~100pt below photo card
  - Alignment: Center
  - Letter-spacing: -0.5pt (tight for elegance)
- **Subtitle**:
  - Font: Roboto Regular or similar sans
  - Size: 13-14pt
  - Case: ALL CAPS
  - Color: White #FFFFFF
  - Position: Centered, ~30pt below title
  - Letter-spacing: 2pt (generous tracking for caps)

## 8. TYPOGRAPHY SYSTEM

### Font Families
- **Serif**: Lora (Regular, SemiBold, Bold)
- **Sans Serif**: Roboto (Regular, Medium, Bold)

### Type Scale

```
Document Title (Cover):        Lora Bold        56-64pt    White on teal
Page Heading:                  Lora Regular     42-48pt    Teal on white
Section Heading:               Lora SemiBold    20-24pt    Teal
Program Name:                  Lora SemiBold    20-22pt    Teal

Body Text (Standard):          Roboto Regular   11-12pt    Black, 1.5× leading
Body Text (Large):             Roboto Regular   13-14pt    Black, 1.5× leading
Label (ALL CAPS):              Roboto Medium    10-11pt    Teal, +1.5pt tracking
Subtitle (Cover):              Roboto Regular   13-14pt    White, +2pt tracking (ALL CAPS)
Caption:                       Roboto Regular   9-10pt     Black or gray

Stat Number:                   Lora Bold        32-36pt    Teal
Stat Label:                    Roboto Regular   10-11pt    Teal (ALL CAPS), +1pt tracking
```

### Typographic Rules
- **Serif for**: Document titles, page headings, section names, program names
- **Sans for**: Body text, labels, captions, contact info
- **ALL CAPS**: Only for labels, cover subtitles, section markers (never body text)
- **Letter-spacing**: +1.5-2pt for ALL CAPS text, -0.5pt for large display titles
- **Leading**: 1.5× for body text (e.g., 11pt → 16.5pt leading)
- **Leading**: 1.2× for display/heading text

## 9. LAYOUT & SPACING SYSTEM

### Page Setup
```
Page Size:          612pt × 792pt (8.5" × 11" US Letter)
Margins:            40pt all sides (top, bottom, left, right)
Text Area:          532pt × 712pt
Bleed (Print):      9pt (3mm) all sides
```

### Grid System
```
Columns:            2-column for programs, 3-column for features
Column Width:       ~260pt (2-col), ~170pt (3-col)
Gutter:             20-30pt
Baseline Grid:      12pt or 6pt
```

### Spacing Scale
```
Extra Large:        60pt       Major section breaks, page-to-page transitions
Large:              40-50pt    Between distinct content blocks
Medium:             32pt       Between related sections
Standard:           20-24pt    Between elements (headings, photos, text)
Small:              12-16pt    Between paragraphs, after headings
Micro:              8pt        Between label and content
```

### Component Spacing
- **Hero photo to title**: 100pt
- **Title to subtitle**: 30pt
- **Heading to divider**: 12pt
- **Divider to body**: 40pt
- **Between programs**: 40-50pt
- **Between stats**: 24pt
- **Logo padding**: 40pt from edges

## 10. IMAGERY GUIDELINES

### Photography Style
- **Tone**: Warm, natural lighting (not studio)
- **Subject**: Authentic people (students, refugees, families, educators)
- **Emotion**: Hope, connection, support, determination
- **Diversity**: Multiple ethnicities, ages, abilities represented
- **Setting**: Real environments (classrooms, homes, outdoors)
- **Avoid**: Generic stock photos, staged corporate imagery, cold lighting

### Photo Treatment
- **Rounded Corners**: 8-24pt radius depending on size
  - Cover photo card: 24pt
  - Supporting images: 8-12pt
- **No Overlays**: Photos appear clean without dark overlays (unlike previous spec)
- **No Borders**: Photos stand alone without strokes
- **Resolution**: 300 DPI minimum for print, 150 DPI for digital

### Photo Placement
- **Cover**: Large centered card, 60-70% page width
- **Hero**: Full-width or near-full at top of page
- **Supporting**: 1/3 to 1/2 column width, integrated with text

## 13. DIFFERENCES FROM PREVIOUS "WORLD-CLASS" SPEC

### What Was Wrong Before

❌ **Not in TFU System**:
- Copper/orange colors (#C87137) - completely absent from real TFU PDFs
- Gold #BA8F5A as primary accent - used very sparingly if at all
- Hero photo overlays with dark gradients - TFU uses clean photo cards
- Stat cards with metrics inline on page 1 - TFU uses sidebar box
- Three-column program cards - TFU uses two-column text layout
- CTA page without Together for Ukraine badge - badge is signature element

✅ **What TFU Actually Does**:
- Pure teal #00393F as dominant color (cover, text, accents)
- Clean rounded photo cards without overlays
- Two-column program listings, not card-based
- Light blue sidebar box for stats (not inline)
- Together for Ukraine badge on closing page
- 3×3 white logo grid on teal background
- Decorative curved divider under major headings

### Design Philosophy
- **TFU is clean, elegant, editorial** - not card-heavy or web-styled
- **Teal dominance** - one primary color used consistently
- **Serif + sans serif contrast** - traditional document design
- **Photography as cards** - images contained in rounded rectangles
- **Minimal decoration** - only the curved divider, no boxes/frames/shadows
