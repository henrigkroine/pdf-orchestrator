# Generated Templates Directory

This directory contains templates generated by the Template Generator system.

## Contents

Generated templates are saved here with three files per template:

1. **`[template-name].json`** - Template specification (data structure)
2. **`[template-name].jsx`** - InDesign script (executable)
3. **`[template-name]-mcp.json`** - MCP payload (automation commands)

## Usage

### Use InDesign Script (.jsx)
```bash
# 1. Open InDesign
# 2. File → Scripts → Other Script...
# 3. Select: [template-name].jsx
# 4. Template created!
```

### Use with Orchestrator
```bash
# 1. Generate template
node template-generator.js generate partnershipBrochure my-template

# 2. Reference in job file
{
  "templateId": "my-template",
  ...
}

# 3. Run orchestrator
node orchestrator.js example-jobs/my-job.json
```
