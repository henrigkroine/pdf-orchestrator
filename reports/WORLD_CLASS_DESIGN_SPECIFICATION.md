## PART 1: MANDATORY BRAND COMPLIANCE

### 1.1 Official Color Palette (ABSOLUTE REQUIREMENTS)

**PRIMARY COLORS** (80% usage):
```
Nordshore: #00393F (deep teal)
  - RGB: 0, 57, 63
  - CMYK: 100%, 10%, 0%, 75%
  - Usage: Headlines, primary accents, CTA buttons, key metrics borders
  - Contrast ratio with white: 12.5:1 (AAA compliant)
```

**SECONDARY COLORS** (20% usage):
```
Sky: #C9E4EC (light blue accent)
  - RGB: 201, 228, 236
  - CMYK: 15%, 0%, 3%, 7%
  - Usage: Background panels, subtle highlights, divider accents

Sand: #FFF1E2 (warm neutral)
  - RGB: 255, 241, 226
  - CMYK: 0%, 5%, 11%, 0%
  - Usage: Warm backgrounds, card backgrounds, section breaks

Beige: #EFE1DC (soft neutral)
  - RGB: 239, 225, 220
  - CMYK: 0%, 6%, 8%, 6%
  - Usage: Alternate backgrounds, subtle texture areas
```

**ACCENT COLORS** (Sparingly):
```
Moss: #65873B (natural green)
  - RGB: 101, 135, 59
  - CMYK: 53%, 25%, 100%, 15%
  - Usage: Program category highlights, success indicators

Gold: #BA8F5A (warm metallic)
  - RGB: 186, 143, 90
  - CMYK: 24%, 40%, 65%, 10%
  - Usage: Key metrics, premium indicators, achievement badges

Clay: #913B2F (rich terracotta)
  - RGB: 145, 59, 47
  - CMYK: 30%, 80%, 80%, 35%
  - Usage: Urgent indicators, important callouts
```

**FORBIDDEN COLORS**:
- ❌ Copper/Orange (#C87137 or similar) - NOT in TEEI palette
- ❌ Pure black text on colored backgrounds - use Nordshore instead
- ❌ AWS orange EXCEPT in AWS logo itself
- ❌ Any colors not listed above

**Accessibility Requirements**:
- Minimum contrast ratio: 4.5:1 (WCAG 2.2 AA)
- Nordshore on white: 12.5:1 ✅
- Gold on white: 4.8:1 ✅
- Sky on white: 2.1:1 ❌ (use for backgrounds only, never for text)

### 1.3 Layout & Spacing Standards

**GRID SYSTEM**:
```
Columns: 12-column grid
Gutter width: 20pt
Page margins: 40pt all sides (top, bottom, left, right)
Safe zone: 50pt from all edges (for critical content)
Bleed: 3mm (9pt) for print versions
```

**SPACING SCALE** (Consistent rhythm):
```
Extra Large: 60pt - Section breaks, major transitions
Large: 40pt - Between distinct content blocks
Medium: 30pt - Related content groups
Standard: 20pt - Between elements (images, text blocks)
Small: 12pt - Between paragraphs
Micro: 6pt - Between list items
```

**VISUAL HIERARCHY**:
```
Level 1 (Most Important):
  - Size: Largest (42-48pt)
  - Weight: Bold
  - Color: Nordshore (or contrasting white)
  - Position: Top third of page, centered or left-aligned

Level 2 (Supporting):
  - Size: Medium (28-32pt)
  - Weight: Semibold
  - Color: Nordshore
  - Position: Clear visual separation from Level 1

Level 3 (Details):
  - Size: Standard (11-14pt)
  - Weight: Regular
  - Color: Black or Gray
  - Position: Supporting Level 1 and 2
```

**LOGO CLEARSPACE**:
```
Minimum clearspace = Height of logo icon element
Example: If TEEI logo icon is 40pt tall, maintain 40pt clear space on all sides
No text, graphics, or other logos within this boundary
Apply same rule to AWS logo
```

### 1.4 Photography Requirements

**MANDATORY**: Every partnership document MUST include 2-5 authentic photographs showing TEEI programs in action.

**Photography Specifications**:
```
Resolution: 300 DPI minimum for print, 150 DPI for digital
Format: JPEG or PNG (no GIFs, no low-quality stock)
Color grading: Warm tones aligned with Sand/Beige palette
Lighting: Natural light preferred, avoid harsh flash
Subjects: Students, educators, learning environments
Authenticity: Real program moments (never generic stock photos)
Diversity: Represent multiple ethnicities, ages, abilities
Emotion: Connection, hope, engagement (not passive poses)
```

**Photography Treatment**:
```
HERO IMAGES (Full-width backgrounds):
  - Dimensions: Full page width × 40-60% page height
  - Overlay: Nordshore #00393F at 40-60% opacity (gradient from bottom)
  - Text on overlay: White, high contrast
  - Purpose: Emotional connection, set tone

SUPPORTING IMAGES (Content sections):
  - Dimensions: 1/3 to 1/2 page width
  - Treatment: Rounded corners (8pt radius, modern feel)
  - Border: Optional 2pt Nordshore border
  - Caption: 9pt Roboto Flex, Gray #666666, below image
  - Purpose: Illustrate programs, show impact

METRIC CARDS (Impact data):
  - Background photo: Subtle, low opacity (20-30%)
  - Overlay: Solid color (Sky, Moss, or Sand at 90% opacity)
  - Purpose: Visual interest without competing with data
```

**Photography Sourcing Priority**:
1. **Use existing TEEI program photos** (Together for Ukraine, AWS partnership)
2. **Request new photography** from program coordinators
3. **Commission photography** if needed (follow guidelines page 15-16)
4. **Never use**: Generic corporate stock, overly posed shots, low-resolution images
