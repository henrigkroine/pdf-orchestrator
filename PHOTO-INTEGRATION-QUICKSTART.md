# Photo Integration - Quick Start

**Status**: Ready to execute | **Time**: 5 minutes

## Option 1: Automated (Recommended)

```
1. Open TEEI-AWS document in InDesign
2. File > Scripts > Other Script...
3. Select: add_photos_teei_aws.jsx
4. Click "Open"
```

**Result**: All 3 images placed automatically with brand-compliant positioning.
